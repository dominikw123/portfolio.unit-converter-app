# __Unit Converter App__
A fully responsive and intuitive web application for converting units of length, area, weight, volume, speed, temperature, and currencies.
The project is built with modular components and reusable hooks, focuses on maintainability, and provides a smooth, user-friendly experience across all devices.
Currency conversion uses live rates from the Frankfurter API, and user preferences such as the selected theme is stored in local storage to ensure data is preserved between sessions.
